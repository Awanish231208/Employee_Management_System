# Employee Management System (Hibernate CRUD)

## 📌 Project Overview
Employee Management System is a simple Java console application developed using Hibernate ORM. It performs basic CRUD (Create, Read, Update, Delete) operations on employee records stored in a MySQL database.

## 🚀 Features
- Add Employee
- View All Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee
- Hibernate ORM Integration
- MySQL Database Connectivity

## 🛠️ Technologies Used
- Java
- Hibernate ORM
- MySQL
- Maven
- Eclipse IDE

## ⚙️ Prerequisites
- JDK 8 or above
- MySQL
- Maven
- Eclipse IDE

## 🗄️ Database Configuration
Update the database credentials in `hibernate.cfg.xml`.

```xml
<property name="hibernate.connection.url">jdbc:mysql://localhost:3306/ems</property>
<property name="hibernate.connection.username">root</property>
<property name="hibernate.connection.password">your_password</property>
```

## ▶️ How to Run
1. Clone the repository.
2. Import the project into Eclipse.
3. Configure MySQL database.
4. Update `hibernate.cfg.xml`.
5. Run `App.java`.
